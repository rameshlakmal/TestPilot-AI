# Projects Section

Projects are like some open ended questions with a guide. Users can see the guide about project, overview tasks and then they can download relevant datasets. 


## Feature Overview

* Admins can create projects and set content. Content is similar to lessons but we mainly focus on text and images and no videos as for now. Usually a project has different sections guiding user through the process. Think of this like chapters in a book
* Admins can upload datasets when creating a project. These datasets are displayed in the Project page so users can download them easily
* Once a user complete a project, they can mark it as completed(self marking)
* Completing a project does not marks for the user because it is self evaluated
